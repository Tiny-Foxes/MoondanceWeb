What is Project OutFox?

Project OutFox is a fork of the open source rhythm game engine StepMania. It focuses on modernizing its codebase to improve performance and graphical fidelity, refurbishing aspects of the engine that have been neglected, and to improve and expand its support for other game types and styles.
